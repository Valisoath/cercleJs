- Math.PI : fonction qui returne la valeur pi

return 3.141592653589793

- Math.round(x) : fonction qui returne l'arrondissement de x

exemple: Math.round(4.7)

return 5

- Math.pow(x,y) : fonction qui returne x puissance y

exemple: Math.pow(8, 2);

return 64

- Math.sqrt(x) : fonction qui returne la racine carre de x

exemple: Math.sqrt(9)

return 3

- Math.abs(x) : fonction qui return l'absolue du x

exemple: Math.abs(-4.7);

return 4.7

- Math.min() et Math.max() : minimum et maximum

exemple: Math.min(0, 150, 30, 20, -8, -200);

return -200

- Math.random() : aléatoire

exemple: Math.random() * 11

explication: nombre aléatoire entre 0 et 10
